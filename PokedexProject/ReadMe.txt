==Keys to change what is seen==
* A * ------------------------switch between automated and regular
* Space * --------------------clears any search entry by index number
* 0-9 * ----------------------types into a search value
* Enter * --------------------Searches for and grabs the pokemon
* click/s * ------------------switches the color sceme
* Right/Left Arrow Keys * ----Naviagte between Pokemon
* T * ------------------------Show the Type Matchup Table
* S * ------------------------Switch the color scheme
* H * ------------------------Toggle the Help Menu
* click on image * -----------Opens the link to the pokemon on Bulbapedia
* N * ------------------------Auto goes numerically
* M * ------------------------Auto goes Alphabetically
* R * ------------------------Auto goes randomly (Default)
